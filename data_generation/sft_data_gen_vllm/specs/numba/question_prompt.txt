### Constraints
- The question must be written in English, 1–5 sentences, clear and testable.
- Do NOT mention any concrete API names, class names, or module paths (e.g., `{module_name}.xxx` or `{module_alias}.xxx`).
- The task should be naturally solvable by writing a small CUDA kernel and a small test harness.
- The task should reasonably benefit from using the provided seed APIs. When it is sensible, the solution should use multiple seed APIs for setup, indexing, and execution (do not force unnatural usage).
- Avoid network/file I/O, external dependencies, and randomness.
- Prefer small deterministic array sizes and simple numeric dtypes (e.g., int32/float32).
- Do not require a specific GPU model or advanced CUDA features; tasks should remain runnable in a CUDA-simulator mode.
